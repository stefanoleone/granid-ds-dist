## [1.0.0] — 2026-07-09

Initial published distribution. `granid-web.css` built from commit `f6ad3fc`
— **byte-identical** to the nine layers `granid-website` vendored by hand at
GWEB-71, so the site's swap from the interim vendoring to the published file
is a **zero visual change**.

### Notes
- Bundle: tokens (primitives + semantic) + base reset + editorial + buttons +
  forms + feedback + surfaces + `patterns/marketing.css`, light theme.
- Frozen snapshot: no Layer 3 override hook, no dark block (ADR 0005 D2).
- `manifest.json` ships alongside with `sha256` and an `sha384` SRI
  `integrity` value per bundle.

### Deferred (nice-to-have, tracked for a future release)
- `granid-web.min.css`: needs a real CSS minifier (dep decision); low value
  over gzipped transport. The `integrity` hash already covers the pin.
